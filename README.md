# TaskUp

A modern task management app built with **React** and **TailwindCSS**, designed to help users create, manage, and organize tasks efficiently. Users can categorize, sort, and mark tasks as completedâ€”with optional features like tag filtering and drag & drop task reordering.

##  Features

-  Create, Edit, and Delete tasks  
-  Categorize tasks (e.g., Work, Personal)  
-  Mark tasks as Completed  
-  Sort tasks by Due Date or Priority  
-  (Optional) Tag-based Filtering  
-  (Optional) Drag & Drop Task Reordering  
-  Dark/Light Mode Toggle  
-  Responsive Design with TailwindCSS
